# Database and API Integration Documentation\n\n## Overview\n\nThis document describes the database models and REST API endpoints for the DDoS Attack Prevention System.\n\n## Database Architecture\n\n### 1. User Management (`User` table)\n- Stores user account information\n- Tracks phone numbers, email, location data\n- Flagging system for suspicious users\n- Fields:\n  - `user_id`: Unique identifier\n  - `email`: User email (indexed)\n  - `phone_number`: User phone (indexed for investigation)\n  - `country`, `region`, `address`: Location data\n  - `is_flagged`: Suspicious user flag\n  - `flag_reason`: Reason for flagging\n\n### 2. Browser License (`BrowserLicense` table)\n- Records of all browser installations\n- License status tracking\n- Machine hash for device identification\n- Fields:\n  - `license_id`: Unique license\n  - `user_id`: Associated user\n  - `machine_hash`: Device fingerprint (indexed)\n  - `status`: ACTIVE, SUSPENDED, REVOKED\n  - `cost_paid`: Payment amount\n\n### 3. DDoS Activity (`DDoSActivity` table)\n- Complete attack records\n- Geographic source tracking\n- Attack type and severity classification\n- Fields:\n  - `activity_id`: Unique activity\n  - `user_id`: Attacking user\n  - `ip_address`: Attacker IP (indexed)\n  - `country`, `region`, `city`: Geographic location (indexed)\n  - `latitude`, `longitude`: Precise GPS coordinates\n  - `target_site`: Target URL\n  - `attack_type`: DNS, WEBSERVER, BANDWIDTH, COMBINED\n  - `severity`: LOW, MEDIUM, HIGH, CRITICAL\n  - `duration_seconds`: Attack duration\n  - `request_count`: Number of requests\n\n### 4. Penalty (`Penalty` table)\n- Penalty issuance tracking\n- Links to DDoS activities\n- Payment status\n- Fields:\n  - `penalty_id`: Unique penalty\n  - `user_id`: Penalized user\n  - `ddos_activity_id`: Associated attack\n  - `amount`: Penalty amount (Rs.)\n  - `status`: PENDING, PAID, OVERDUE, WAIVED\n  - `is_paid`: Boolean payment flag\n\n### 5. Payment (`Payment` table)\n- Complete payment history\n- Links to licenses and penalties\n- Payment method tracking\n- Fields:\n  - `payment_id`: Unique payment\n  - `user_id`: Payer\n  - `transaction_id`: Payment gateway reference (indexed)\n  - `amount`: Payment amount\n  - `payment_method`: UPI, DEBIT_CARD, CREDIT_CARD, NET_BANKING\n  - `payment_type`: PURCHASE, PENALTY, RENEWAL\n  - `status`: SUCCESSFUL, FAILED, PENDING\n\n### 6. Cyber Crime Report (`CyberCrimeReport` table)\n- Reports ready for submission to authorities\n- Contains all evidence and user information\n- Government tracking\n- Fields:\n  - `report_id`: Unique report\n  - `user_id`: Suspect user\n  - `activity_id`: Associated DDoS activity\n  - `country`, `cyber_crime_department`: Jurisdiction\n  - `user_name`, `user_phone`, `user_email`: Contact info\n  - `ip_address`, `user_location`: Location data\n  - `attack_details`: Detailed attack information\n  - `bank_account_info`: Financial investigation data (encrypted)\n  - `submission_status`: DRAFT, SUBMITTED, ACKNOWLEDGED\n  - `government_reference_number`: Govt case number\n\n### 7. Geo Location (`GeoLocation` table)\n- IP geolocation cache\n- VPN/Proxy detection\n- ISP information\n- Fields:\n  - `ip_address`: IP (indexed)\n  - `country`, `region`, `city`: Location\n  - `latitude`, `longitude`: GPS coordinates\n  - `isp`: Internet Service Provider\n  - `is_vpn`, `is_proxy`: Detection flags\n  - `threat_level`: LOW, MEDIUM, HIGH\n\n### 8. Attack Statistics (`AttackStatistics` table)\n- Aggregated metrics by country/region\n- Trend analysis\n- Top targets\n\n### 9. Audit Log (`AuditLog` table)\n- Complete system audit trail\n- Compliance and security logging\n- All actions tracked\n\n## REST API Endpoints\n\n### Base URL\n```\nhttp://localhost:8000/api/v1\n```\n\n### User Management Endpoints\n\n**POST /users/register**\n- Register new user\n- Collects: email, phone, name, country, region, address\n- Returns: user_id, registration date\n\n**GET /users/{user_id}**\n- Get user information\n- Returns: all user data\n\n**GET /users/search/email**\n- Search user by email\n- Query: `?email=user@example.com`\n\n**GET /users/search/phone**\n- Search user by phone\n- Query: `?phone=+919876543210`\n\n**PUT /users/{user_id}/flag**\n- Flag user as suspicious\n- Data: `reason` (string)\n\n**GET /users/flagged/list**\n- List all flagged users\n- Returns: flagged users with reasons\n\n### Browser License Endpoints\n\n**POST /licenses/create**\n- Create browser license record\n- Data: user_id, browser_type, installation_id, machine_hash\n- Returns: license_id, purchase_date\n\n**GET /licenses/{license_id}**\n- Get license information\n- Returns: license details and status\n\n**GET /users/{user_id}/licenses**\n- Get all licenses for user\n- Returns: list of licenses\n\n**PUT /licenses/{license_id}/suspend**\n- Suspend a license\n- Data: `reason` (string)\n\n### DDoS Activity Endpoints\n\n**POST /ddos/report-activity**\n- Report DDoS attack\n- Data: user_id, ip_address, country, region, city, target_site, attack_type, severity, duration_seconds, request_count\n- Returns: activity_id, timestamp\n\n**GET /ddos/activity/{activity_id}**\n- Get activity details\n- Returns: complete activity information\n\n**GET /ddos/user/{user_id}**\n- Get all activities for user\n- Query: `?limit=10`\n- Returns: list of user's DDoS activities\n\n**GET /ddos/country/{country}**\n- Get all attacks from country\n- Query: `?limit=50`\n- Returns: attacks originating from country\n\n**GET /ddos/region/{region}**\n- Get all attacks from region\n- Query: `?limit=50`\n- Returns: attacks from region\n\n**GET /ddos/ip/{ip_address}**\n- Get all attacks from IP\n- Returns: all activities from IP\n\n**GET /ddos/critical**\n- Get critical severity attacks\n- Query: `?limit=50`\n- Returns: critical attacks only\n\n### Penalty Endpoints\n\n**POST /penalties/create**\n- Create penalty\n- Data: user_id, reason, amount (default 5000), activity_id\n- Returns: penalty_id, issued_date, due_date\n\n**GET /users/{user_id}/penalties**\n- Get all penalties for user\n- Returns: list of penalties\n\n**GET /users/{user_id}/penalties/pending**\n- Get unpaid penalties\n- Returns: pending penalties and total amount due\n\n**PUT /penalties/{penalty_id}/mark-paid**\n- Mark penalty as paid\n- Returns: success message\n\n### Payment Endpoints\n\n**POST /payments/record**\n- Record payment transaction\n- Data: user_id, transaction_id, amount, payment_method, payment_type, browser_license_id, penalty_id\n- Returns: payment_id, timestamp\n\n**GET /users/{user_id}/payments**\n- Get all payments by user\n- Query: `?limit=50`\n- Returns: payment history\n\n### Cyber Crime Report Endpoints\n\n**POST /cybercrime/report**\n- Create cyber crime report\n- Data: user_id, country, cyber_crime_department, user_name, user_email, user_phone, ip_address, user_location, attack_details, bank_account_info\n- Returns: report_id, submission_status\n\n**GET /cybercrime/country/{country}**\n- Get reports for country\n- Query: `?limit=50`\n- Returns: reports for authorities\n\n**PUT /cybercrime/{report_id}/submit**\n- Submit report to authorities\n- Data: `govt_reference` (government case number)\n- Returns: confirmation with reference number\n\n### Statistics Endpoints\n\n**GET /statistics/attacks**\n- Get attack statistics\n- Query: `?country=India&days=7`\n- Returns: total attacks, duration, requests, countries, critical attacks\n\n**GET /statistics/revenue**\n- Get total revenue\n- Returns: total_revenue, currency\n\n### Health Check\n\n**GET /health**\n- API status check\n- Returns: status, version, timestamp\n\n## Usage Examples\n\n### Register User\n```bash\ncurl -X POST \"http://localhost:8000/api/v1/users/register\" \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n
